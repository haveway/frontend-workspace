CSS 기본 선택자

선택자란?

특정 html요소를 선택하고자 할 때 사용하는 기능
해당 요소를 선택해서 거기에만 원하는 "스타일"을 적용 가능

1. 모든(전체) 선택자 : * 

        [ 표현법 ]
        * {
            스타일 속성 : 스타일;
            스타일 속성 : 스타일;
        }

        => 모든 요소들을 다 선택하고자 할때 사용

2. 태그 선택자(태그명)

        [ 표현법 ]
        태그명 {
            스타일 속성 : 지정할 내용;
            ...
        }
        => 해당 이 문서 내의 해당 같은 태그들을 모두 선택할 때 사용

[ 표현법 ] 태그명 { 스타일 속성 : 지정할 내용; ... } => 해당 이 문서 내의 해당 같은 태그들을 모두 선택할 때 사용

3. 아이디 선택자 (#아이디명)

        [ 표현법 ]
        # 아이디명{
            스타일 속성 : 지정할 내용;
            ...
        }
        => 해당 이 문서 내의 특정 html 요소 하나만을 선택하고자 할 때 사용
        단, 해당 그 요소에 id속성을 이용해서 고유한 아이디값을 부여해야함
        같은 태그더라도 부분적으로 다르게 스타일을 부여하고 싶을 때 사용하면 좋음

4. 클래스 선택자(.클래스명)

        [ 표현법 ]
        .클래스명{
            스타일 속성 : 지정할 내용;
            ...
        }
        => 해당 이 문서 안에 내가 원하는 여러개의 요소를 선택할 때 사용

동위(같은레벨) 선택자

동위 관계(같은레벨)에 있는 뒤에 위치한 특정 요소를 선택할 때 사용 

        [표현법]
        - a요소 뒤에있는 b요소 "단 하나만을" 선택
        a + b{
            스타일 속성 : 스타일;
            ....
        }

        - a요소 뒤에 있는 b요소들을 "전부 다 " 선택
        a ~ b{
            스타일 속성 : 스타일;
            ....
        }
